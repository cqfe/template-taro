# 移动端模板

## 项目简介

移动端模板，采用 TaroJs@4.x，VueJs@3.x，NutUi@4.x 开发。

## 开发规范

[前端开发规范](https://ones.cisdigital.cn/wiki/#/team/SKG3mSQb/space/A8ZckFsc/page/Jj69ks48)

## 开发启动方式

```bash
# 安装依赖
pnpm i
npm run dev:h5
```

## 生产构建方式

```bash
npm run build
```
